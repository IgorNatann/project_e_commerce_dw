# Status Reports

Relatorios periodicos para equipe e cliente, com historico por data.

## Como Gerar

Relatorio do dia atual:

python scripts/status_reports/generate_status_report.py

Relatorio para uma data especifica:

python scripts/status_reports/generate_status_report.py --date 2026-02-28

Sobrescrever um relatorio existente:

python scripts/status_reports/generate_status_report.py --date 2026-02-28 --force
